Spring Data JPA介绍

持久化操作：内存中的数据模型转换为存储模型，以及将存储模型转换为内存中的数据模型的统称。一般是数据库操作。

JPA：Java Persistence API，是Sun官方提出的Java持久化规范。规范，即只定义标准规则，不提供实现。

Spring Data JPA是Spring Data 的一个子项目，通过提供基于JPA的Respository，极大地减少了JPA作为数据访问方案的代码量。

通过Spring Data JPA框架，开发者可以省略实现持久层业务逻辑的公国，唯一要做的就是声明持久层的接口，其他都交给Spring Data JPA 来完成。
