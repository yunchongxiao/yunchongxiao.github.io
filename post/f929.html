统计学习及监督学习概论

Created2023-04-02|Updated2023-04-02|统计学习方法 （第二版）
|Post View:

统计学习概念

计算机基于数据构建概率统计模型，并运用模型对数据进行预测

  • 以计算机和网络为平台
  • 以数据为研究对象
  • 以预测和分析数据为目的
  • 以方法为中心
  • 是多领域交叉的学科

统计学习方法步骤

  1. 得到有限的训练数据集
  2. 确定学习模型的集合（假设空间）
  3. 确定模型选择的准则（策略）
  4. 实现求解最优模型的算法（算法）
  5. 通过学习方法选择最优模型
  6. 利用该模型对新数据进行预测分析

统计学习分类

基本分类

  • 输入空间：输入的所有取值的集合
  • 特征空间：所有特征向量存在的空间
    • 输入向量通过核函数映射为特征向量
  • 输出空间：输出的所有可能的取值的集合

  • x(i)x^{(i)}x(i): 列向量 xxx 的第 iii 个特征
  • xix_ixi​: 第 iii 个输入列向量
  • 训练集：{(x1,y1),…,(xN,yN)}\{(x_1, y_1), \dots, (x_N, y_N)\}{(x1​,y1​),…,(xN​,yN​)}
  • 基本假设：X,YX,YX,Y具有联合概率密度分布P(X,Y)P(X, Y)P(X,Y)
  • 模型形式：
    • 条件概率分布P(Y∣X)P(Y|X)P(Y∣X)

  • 假设空间：所有模型的集合

无监督学习

  • 定义：从无标注数据中学习预测模型，本质为学习数据中的统计规律或潜在结构

  • 输入空间：XXX

  • 隐式结构空间：ZZZ

    • 条件概率分布P(z∣x)P(z|x)P(z∣x)或P(x∣z)P(x|z)P(x∣z)

  • 假设空间：所有可能的模型的集合

  • 目的：选出在给定评价标准下的最优模型

  • 训练集：U={x1,…,xN}U= \{x_1, \dots, x_N\}U={x1​,…,xN​}

其他

  • 强化学习：智能系统与环境不断交互，每次行动得到奖惩分数
  • 半监督学习：仅有少量数据有标注
  • 主动学习：主动选择有用的实例进行标注

模型分类

  • 概率模型、非概率模型
  • 线性模型、非线性模型
  • 参数化模型、非参数化模型

算法分类

  • 在线学习，来一个学一个
  • 批量学习，一次学一批

技巧分类

  • 贝叶斯学习，基于贝叶斯定理
  • 核方法，基于核函数

统计学习三要素

模型+策略+算法

模型

  • 线性模型或非线性模型，模型中的参数可以代表模型，如y = ax+b，则a, b 就可以代表模型

策略

  • 损失函数：度量模型一次预测的好坏，记作L(Y,f(X))L(Y, f(X))L(Y,f(X))
    • 0-1损失函数：Y==f(X)?0:1Y == f(X)? 0: 1Y==f(X)?0:1
    • 平方损失函数：[Y−f(X)]2[Y-f(X)]^2[Y−f(X)]2 (评估决策函数)
    • 绝对损失函数：∣Y−f(X)∣|Y-f(X)|∣Y−f(X)∣ (评估决策函数)
    • 对数损失函数：−logP(Y∣X)-log^{P(Y|X)}−logP(Y∣X) (评估条件概率分布，负号保证损失函数为正数)
  • 风险函数：E(L(Y,f(X)))E(L(Y, f(X)))E(L(Y,f(X)))，度量平均意义下模型预测的好坏，本质是对损失函数求期望
  • 经验风险：∑L(Y,f(X))/N\sum L(Y, f(X))/N∑L(Y,f(X))/N，损失函数的平均值，本质是代替难以计算的风险函数
  • 结构风险：∑L(Y,f(X))/N+λJ(f)\sum L(Y, f(X))/N + \lambda J(f)∑L(Y,f(X))/N+λJ(f)，经验风险+惩罚项，本质是防止过拟合，因为经验风险最小的模型往往是过拟合的

最终我们追求结构风险最小化

算法

存在解析解，则直接计算得出，否则通过数值分析方法得出，如梯度下降

# 模型评估

  • 训练误差：模型对于训练数据的预测能力
  • 测试误差：模型对于测试数据的预测能力
  • 过拟合：本质是模型包含的参数过多

模型选择

  • 正则化：结构风险 ∑L(Y,f(X))/N+λJ(f)\sum L(Y, f(X))/N + \lambda J(f)∑L(Y,f(X))/N+λJ(f) 最小化策略
    • L1范数：λJ(f)=∣∣w∣∣1\lambda J(f) = ||w||_1λJ(f)=∣∣w∣∣1​，就是参数的绝对值和，多用于筛选参数，因为w可以为0
    • L2范数：λJ(f)=∣∣w∣∣2\lambda J(f) = ||w||_2λJ(f)=∣∣w∣∣2​，就是参数的平方和，多用于防止过拟合
    • 奥卡姆剃刀原理：选择能很好解释的，参数很简单的模型
  • 交叉验证：
    • 简单交叉验证：将数据集随机划分为两部分，即训练集和测试集
      • K折交叉验证：将数据集划分为 K 个互不相交，大小相同的子集，使用一份作为测试集，其余9份作为训练集，可以得到 K 个模型
    • 留一交叉验证：仅留一个数据为测试集

泛化能力

  • 模型对于未知数据的预测能力，（测试集是已知数据集）
  • 泛化误差：对于整个数据集的风险函数
  • 泛化误差上界：
  • Hoeffding 不等式，待定证明

生成模型

  • 定义：由数据学习得到联合分布概率P(X,Y)P(X, Y)P(X,Y)，然后求出P(Y∣X)P(Y|X)P(Y∣X)作为预测模型（即生成模型 Generative Model）

    P(Y∣X)=P(X,Y)P(X)P(Y|X) = \frac{P(X, Y)}{P(X)} P(Y∣X)=P(X)P(X,Y)​

  • 经典生成模型

    • 朴素贝叶斯
    • 隐马尔科夫模型（时间序列模型）

判别模型

  • 有数据学习得到决策函数f(X)f(X)f(X)或直接得到P(Y∣X)P(Y|X)P(Y∣X)作为预测模型（即判别模型 Discriminative Model）
  • 典型的判别模型：
    • k 邻近法、感知机、

生成模型 vs 判别模型

  • 生成模型：模型需要识别出当前动物是狗还是大象
  • 判别模型：只需要识别出鼻子，就可以通过鼻子长短进行判别

生成模型：

  • 所需数据量大
  • 可得到联合概率分布
  • 收敛速度更快
  • 能反映同类数据的特性
  • 隐变量存在时，仍可用生成模型

判别模型

  • 数据量相比生成模型较少
  • 面向预测的算法，准确率更好
  • 可以简化学习问题
  • 不能反应数据本身特性

分类问题

  • 混淆矩阵

    真实正类 真实负类
    预测正类 True Positive False Positive
    预测负类 False Negative True Negative

  • 精确率：预测的准不准，所以是所有预测为正类的样本中，正确预测的比例 TPTP+FP\frac{TP}{TP+FP}TP+FPTP​

  • 召回率：真实的准不准，所以是所有真实为正类的样本中，正确预测的比例 TPTP+FN\frac{TP}{TP+FN}TP+FNTP​

  • 调和值：精确率与召回率的调和平均

  • 方法

    • 感知机
    • k 邻近算法
    • 朴素贝叶斯
    • 决策树
    • 逻辑回归

标注问题

  • 定义：输入变量为序列变量，即当前变量与之前的变量可能有关

  • 方法

    • 隐马尔科夫模型
    • 条件随机场

  • 应用

  • 从一个句子中提取名字短语，每个单词可同时拥有（B, E, O）三个标记

    • At 的标记为 O
    • Microsoft 的标记为 B
    • desire 的标记为 BE

回归问题

  • 类型：

    • 一元回归、多元回归
    • 线性回归、非线性回归

  • 损失函数：平方损失
