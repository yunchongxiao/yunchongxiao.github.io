5. Longest Palindromic Substring

Given a string s, return the longest palindromic substring in s.

Example 1:

Input: s = "babad"
Output: "bab"
Explanation: "aba" is also a valid answer.

Example 2:

Input: s = "cbbd"
Output: "bb"

Constraints:

  • 1 <= s.length <= 1000
  • s consist of only digits and English letters.

Dynamic Programming:

  1. 数据规模：s.length <= 1000，使用二维数组不会溢出，则状态可以考虑使用二维数组定义，即dp[i][j]，下面需要思考dp[i][j]的含义

  2. 本题的最终输出结果为最长的回文子串，对于回文串而言，我们想要扩展其规模，需要在当前字符串是回文串的基础上，左右两侧都添加相同的字符。因此我们需要清晰的知道该串的两侧边界，因此我们的状态dp[i][j]必须表示该串s[i:j]的信息，因此我们也变相的得到了子串的长度，即j-i+1，因此，我们的dp[i][j] 只需要表示该串是否为回文串即可。

  3. 状态转移方程：s[i:j]如果为回文串，必须满足两端字符相同s[i] == s[j]，并且子串也是回文串dp[i+1][j-1] == true，即dp[i][j] = dp[i+1][j-1] && s[i] == s[j]

  4. 状态转移顺序：状态为dp[i][j]，因此显然是二重循环，我们的转移，较长的回文串依赖较短的回文串，因此我们的最外层循环应该表示长度，每次内循环处理的较短长度的回文串转移，当前长度的所有状态都已经得到后，我们才可以继续处理更长的字符串。

  5. 初始化：我们可以考虑几个较短的回文串： ,a, aa, aba, abba

    • 长度为0：空串应该被视作回文串，因为满足将两端加入相同的字符后，仍是回文串，即dp[i][i-1] = true
    • 长度为1：一定是回文串，因此直接初始化，则dp[i][i+1] = true
    • 长度为2：从空串的状态转移得到
    • 长度为3：是通过长度为 1 的状态转移得到的

    因此我们只需要初始化长度为 0 和 1 的状态即可

  6. 全局记录变量：题目需要的结果不一定为dp[i][j]，可能出现在中间某一步，因此需要全局记录

class Solution {
public:
  string longestPalindrome(string s) {
    int start = 0; // the start of the substring
    int maxLength = 1; // the length of the substring
    const int MAXN = 1e3+10;
    int n = s.size();
    vector<vector<int>> dp(n, vector<int>(n));
    // initialize
    for (int i = 0; i < n; i++) dp[i][i] = true;
    for (int i = 0; i < n-1; i++) dp[i+1][i] = true;
    for (int length = 2; length <= n; length++) {
      for (int i = 0; i < n; i++) {
        // length = j-i+1 -> j = length+i-1
        int j = length + i - 1;
        if (j >= s.size()) break; // invalid
        // s[i:j] is palindrome
        if (dp[i+1][j-1] && s[i] == s[j]) {
          dp[i][j] = true;
          if (maxLength < length) {
            start = i; maxLength = length;
          }
        }
      }
    }
    return s.substr(start, maxLength);
  }
};
  • 时间复杂度：O(n2)O(n^2)O(n2)
  • 空间复杂度：O(n2)O(n^2)O(n2)
