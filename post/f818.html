53. Maximum Subarray

Given an integer array nums, find the subarray with the largest sum, and return its sum.

Example 1:

Input: nums = [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
Explanation: The subarray [4,-1,2,1] has the largest sum 6.

Example 2:

Input: nums = [1]
Output: 1
Explanation: The subarray [1] has the largest sum 1.

Example 3:

Input: nums = [5,4,-1,7,8]
Output: 23
Explanation: The subarray [5,4,-1,7,8] has the largest sum 23.

Constraints:

  • 1 <= nums.length <= 10^5
  • -104 <= nums[i] <= 10^4

analysis:

  1. 数据规模：1e5，只能用一维数组，因此考虑 dp[i] 的含义
  2. 单参数 dp[i] ，一般表示nums[i]为末尾的相关信息，这里表示以 nums[i] 结尾的所有子数组中，最大和的值
  3. 状态转移：当我们希望计算 dp[i] 时，我们已知 nums[i] 和 dp[i-1]，因为 dp[i] 表示以 nums[i] 为结尾的子数组，因此一定包含 nums[i]，对于之前的元素，我们知道其最大和为 dp[i-1]，如果该和大于零，则我们的 nums[i] 就与之一起成组，如果该和小于等于零，我们的 nums[i] 就自成一组。
  4. 初始化：dp[0] = nums[0]
  5. 其他细节，最终答案不一定为 dp[n] ，因此我们需要全局变量记录全局最大值

Code:

class Solution {
public:
  int maxSubArray(vector<int>& nums) {
    int n = nums.size();
    vector<int> dp(n);
    int maxLength = 1;
    dp[0] = nums[0];
    for (int i = 1; i < n; i++) {
      dp[i] = nums[i] + (dp[i-1] > 0? dp[i-1] : 0);
      maxLength = max(dp[i], maxLength);
    }
    return maxLength;
  }
};
