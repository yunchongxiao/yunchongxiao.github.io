安装 Anaconda

Anaconda 官网下载地址

  1. 安装完毕后打开 Anaconda Powershell Prompt

  2. 创建python运行环境并激活

    conda create --name d2l python=3.9 -y
conda activate d2l

  3. 安装 pytorch和tensorflow

    pip install torch
pip install torchvision

pip install tensorflow
pip install tensorflow-probability

  4. 安装本书的 d2l 包，以方便调取本书中经常使用的函数和类：（好像有冲突）

    pip install d2l

  5. 测试是否安装成功

引入 tensorflow 时报错，可能是没有安装 cuda，

打开NVIDIA Control Panel 查看显卡版本

在上方的帮助中知道系统信息，查看显卡版本

在官网对照，找到对应的 CUDA 版本

下载对应的 CUDA

根据自己的操作系统选择合适的版本
