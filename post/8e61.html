React main concepts（二）

|字数总计:886|阅读时长:3分钟|阅读量:|评论数:

建议配合官网文档 Introducing jsx 章节进行阅读。

Introducing jsx

jsx 是 JavaScript XML 的简写，其实是函数调用和表达的语法糖，最终会转化为 JavaScript 代码。 如下例中变量 element 所保存的像是字符串又或者是HTML的奇怪的标签 <h1>hello, world!</h1> 便是 jsx 。

const element = <h1>Hello, world!</h1>;

Why jsx?

  1. 传统的网页开发，一般会人为的将整个网页分为 html 、 css 、 js 三个部分，但这会各个业务逻辑的代码是分布在各个地方，这对于编码而言是不方便的，因此我们可以通过 jsx 将这三个部分放在一起，从而简化代码编程的难度。

  2. jsx 不是 React 所必须的，但使用 jsx 可以极大的优化代码的观感，因此基本上 React 中均使用 jsx 语法。

Embedding Expressions in jsx

在下面的的例子中，我们首先创建了一个名为 name 的变量，并通过花括号 {} ，在 jsx 表达式中进行引用：

const name = "Chong Jiang";
const element = <h1>hello, {name}</h1>

也就是说，在 jsx 表达式中，我们可以通过花括号 {} ，在 jsx 表达式中插入任意合法的 JavaScript 表达式，比如 2+2 、 user.firstName 、 formatName(user) 等等。如下例所示：

function formatName(user) {
  return user.firstName + ' ' + user.lastName;
}

const user = {
  firstName: 'Harper',
  lastName: 'Perez'
};

const element = (
  <h1>
    Hello, {formatName(user)}!
  </h1>
);

需要注意的是，由于 JavaScript 在编译时会自动补全缺失的句尾分号 ; ，因此为了避免 jsx 分号后被分号 ; 错误的分割，我们需要在 jsx 表达式外使用小括号 () 进行包裹。

jsx is an Expression Too

jsx 也是一个表达式，因此我们可以在函数中返回 jsx 或者在 if 语句的条件表达式以及其他条件中使用 jsx

function getGreeting(user) {
  if (user) {
    return <h1>Hello, {formatName(user)}!</h1>;
  }
  return <h1>Hello, Stranger.</h1>;
}

Specifying Attributes with jsx

我们可以给 jsx 表达式赋予属性，常见的属性值有两种：字符串以及 JavaScript 表达式。

  1. 字符串：

    const element = <div tabIndex="0"></div>;

  2. JavaScript 表达式：

    const element = <img src={user.avatarUrl}></img>;

注意，与 JavaScript 的语法不同，当属性值为字符串时，使用引号，用来表示属性值为字符串；当属性值为 JavaScript 表达式时，使用大括号，表示 JavaScript 表达式（Embedding Expressions in jsx），不可既存在引号又存在大括号。

Specifying Children with jsx

jsx标签可以包含许多子标签：

const element = (
  <div>
    <h1>Hello!</h1>
    <h2>Good to see you here.</h2>
  </div>
);

如果标签不需要内容，我们可以写成自闭合的形式：

const element = <img src={user.avatarUrl} />;

jsx Represents Objects

Babel 会将 jsx 编译成 React.createElement()并调用，因此下面两种写法是完全等价的：

const element = (
  <h1 className="greeting">
    Hello, world!
  </h1>
);
const element = React.createElement(
  'h1',
  {className: 'greeting'},
  'Hello, world!'
);

本质上是创建了一个 React 对象：

// Note: this structure is simplified
const element = {
  type: 'h1',
  props: {
    className: 'greeting',
    children: 'Hello, world!'
  }
};

我们可以把 React 对象看作是对于页面上想呈现的内容的一种描述，React 可以通过这种描述创建 DOM 并自动更新。

Babel 是一个 JavaScript 编译器，可以将 ES6 语法的代码转化为向后兼容的 JavaScript 代码，也可以将 jsx 转化为 React 语法

我们建议使用“Babel” 提供的语言定义 插件来正确的高亮显示 ES6 和 jsx 代码。
