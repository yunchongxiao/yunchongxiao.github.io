10. Regular Expression Matching

Given an input string s and a pattern p, implement regular expression matching with support for '.' and '*' where:

  • '.' Matches any single character.
  • '*' Matches zero or more of the preceding element.

The matching should cover the entire input string (not partial).

Example 1:

Input: s = "aa", p = "a"
Output: false
Explanation: "a" does not match the entire string "aa".

Example 2:

Input: s = "aa", p = "a*"
Output: true
Explanation: '*' means zero or more of the preceding element, 'a'. Therefore, by repeating 'a' once, it becomes "aa".

Example 3:

Input: s = "ab", p = ".*"
Output: true
Explanation: ".*" means "zero or more (*) of any character (.)".

Constraints:

  • 1 <= s.length <= 20
  • 1 <= p.length <= 20
  • s contains only lowercase English letters.
  • p contains only lowercase English letters, '.', and '*'.
  • It is guaranteed for each appearance of the character '*', there will be a previous valid character to match.

Analysis:

  • 本题的关键在于如何处理通配符 * ，由于通配符*通配上一个字符，因此我们如果从左向右考虑的话，则对于每一个当前字符，都需要判断其后一个字符是否为*，因此我们考虑从右向左处理
  • 数据规模：20，因此可以使用二维数组，则状态的形式定义为 dp[i][j]
  • 由于我们有主串 s 和 模式串 p，因此常见的状态定义方法为 dp[i][j]表示s[i:-1]能否用 p[j:-1]匹配
  • 状态转移：欲求dp[i][j]时，我们的已知量为 s[i]，p[j]以及dp[i+1][j+1], dp[i][j+1], dp[i+1][j]
    1. p[j] 为 . 则 p[j]一定能和 s[i]匹配，则dp[i][j] = dp[i+1][j+1]
    2. p[j] 为普通字符，dp[i][j] = dp[i][j] && p[j+1] == s[i+1]
    3. p[j] 为*，则根据dp[i][j]的定义，一定有dp[i][j] = false，因为 *不能单独使用，这里设置一个flag = true表示下一个字符可以使用 0 - n 次
    4. flag = true 时
      • 当前字符匹配主串的 0 个字符，dp[i][j] = dp[i][j+2]
      • 当前字符匹配主串的 k 个字符，可以看成先匹配 1 个，再匹配 k-1 个，匹配 1 个时，即dp[i][j] = dp[i+1][j+2] && s[i] == p[j]，假如匹配失败，则没有继续匹配的意义；若匹配成功，也没有必要继续了，只要我们让内循环为 主串 s 的规模不断向左扩大，则就相当于保持当前 p 不变，一直尝试多个字符匹配
  • 初始化：主串为空时，长度为1的子串无法匹配
class Solution {
public:
  bool isMatch(string s, string p) {
    int sSize = s.size(), pSize = p.size();
    vector<vector<bool>>(sSize+1, vector<bool>(pSize+1));
    // initialize
    // pattern is "", then can't match any s which length not zero
    for (int i = 1; i <= sSize; i++) dp[i][pSize] = false;
    dp[0][0] = true; // both of them is "" is definited true
    flag = false; 
    for (int j = pSize-1; j >= 0; j--) {
      for (int i = sSize-1; i >= 0; i--) {
        char curP = p[j], curS = s[i];
        if (flag == false) {
          if (curP == '.') {dp[i][j] = dp[i+1][j+1];}
          else if (curP == '*') {
            dp[i][j] = false;
        		flag = true;
          }
          else { dp[i][j] = (dp[i+1][j+1] && curP == curS);}
        }
        else {
          if (dp[i][j+2]) {
            dp[i][j] = true;
          }
          else {
            if (curP = '.') {
              dp[i][j] = dp[i+1][j+2];
            }
            else {
              dp[i][j] = (dp[i+1][j+2] && curP == curS);
            }
          }
        }
      }
    }
    return dp[0][0];
  }
};
