Conditional Rendering

在 React 中，我们可以创建不同的组件来封装不同的行为，并根据当前的状态选择显示其中的一些组件。

React 的条件渲染和 JavaScript 的条件语句是类似的，使用if等条件操作符来判断当前状态，并更新 UI 以符合预期。

考虑一下两个组件：

function UserGreeting(props) {
  return <h1>Welcome back!</h1>;
}

function GuestGreeting(props) {
  return <h1>Please sign up.</h1>;
}

接下来，我们要创建一个Greeting组件，来根据用户的登陆状态选择显示userGreeting还是GuestGreeting：

function Greeting(props) {
  const isLoggedIn = props.isLoggedIn;
  if (!isLoggedIn) {
    return <GuestGreeting />
  }
  else {
    return <UserGreeting />
  }
}
ReactDOM.render(<Greeting />, ducument.getElementById('root'));

Greeting组件会根据用户的状态来选择显示不同的组件。

Element Variables

我们可以使用变量来保存一些元素，这可以在保证一部分元素不变的情况下根据条件渲染其他元素。
考虑以下登入、登出按钮：

function LoginButton(props) {
  return (
    <button onClick={props.onClick}>
      Login
    </button>
  );
}

function LogoutButton(props) {
  return (
    <button onClick={props.onClick}>
      Logout
    </button>
  );
}

我们可以通过引入state创建一个LoginControl组件，将上面的登入、登出，以及两个打招呼组件，封装在一起：

function UserGreeting(props) {
  return <h1>Welcome back!</h1>;
}

// 登陆前的欢迎组件
function GuestGreeting(props) {
  return <h1>Please sign up.</h1>;
}

// 根据参数选择显示是否登录的欢迎组件
function Greeting(props) {
  const isLoggedIn = props.isLoggedIn;
  if (!isLoggedIn) {
    return <GuestGreeting />
  }
  else {
    return <UserGreeting />
  }
}

// 登录按钮组件
function LoginButton(props) {
  return (
    <button onClick={props.onClick}>
      Login
    </button>
  );
}

// 登出按钮组件
function LogoutButton(props) {
  return (
    <button onClick={props.onClick}>
      Logout
    </button>
  );
}

// 总封装组件
class LoginControl extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      isLoggedIn: false,
    };
  }

  handleLoginClick = () => {
    // 登录的逻辑代码...
    this.setState({isLoggedIn: true});
  }

  handleLogoutClick = () => {
    // 登出的逻辑代码...
    this.setState({isLoggedIn: false});
  }

  render() {
    let isLoggedIn = this.state.isLoggedIn;
    let button;
    if (isLoggedIn) {
      button = <LogoutButton onClick={this.handleLogoutClick}/>;
    }
    else {
      button = <LoginButton onClick={this.handleLoginClick}/>;
    }
    return (
      <div>
        <Greeting isLoggedIn={this.state.isLoggedIn}/>
        {button}
      </div>
    )
  }
}

ReactDOM.render(<LoginControl />, document.getElementById('root'))

Inline If with Logical && Operator

你可以在 JSX 里通过大括号嵌入表达式，这可以很方便的使你有条件的加入一个元素：

function Mailbox(props) {
  const unreadMessage = props.unreadMessage;
  return (
    <div>
      <h1>Hello!</h1>
      {unreadMessage.length > 0 &&
        <h2>
          You have {unreadMessage.length} unread messages.
        </h2>
      }
  )
}
// 模拟未读数据
const messages = ['react', 'Re: react', 'Re: Re: React']；
ReactDOM.render(<Mailbox unreadMessage={messages}/>, document.getElementById('root'));

这是因为，在 JavaScript 中，false && expression 总是返回 false， true && expression总是返回expression，我们可以利用这个特性有条件的返回满足条件的元素。

Inline If-Else with Conditional Operator

另一种条件渲染是使用 JavaScript 中的三目运算符condition ? true : false，在下面的例子中，我们通过三目运算符选择性的显示文本：

render() {
  const isLoggedIn = this.state.isLogedIn;s
  return (
    <div>
      The user is {isLoggedIn ? "already" : "not"} log in.
    </div>
  );
}

  const isLoggedIn = this.state.isLoggedIn;

Preventing Component from Rendering

当你不希望一个组件被渲染时，你可以返回false来阻止其渲染：

unction WarningBanner (props) {
// 当参数中 warn 为 false 时，返回 null 表示不渲染
if (!props.warn) {
  return null;
}

return (
  <div className="warning">
    Warning!
  </div>
)
}

class Page extends React.Component {
constructor() {
  super();
  this.state = {showWarning: true};
}

handleToggleClick = () => {
  this.setState(prevState=>({
    showWarning: !prevState.showWarning
  }));
}

render() {
  return (
    <div>
      <WarningBanner warn={this.state.showWarning} />
      <button onClick={this.handleToggleClick}>
        {this.state.showWarning ? "Hide": "Show"}
      </button>
    </div>
  )
}
}

ReactDOM.render(<Page />, document.getElementById('root'));

即使返回null，即不渲染组件，组件所定义的证明周期函数也会正常运行。
