MySQL安装

下载免安装MySQL包

  1. 进入 MySQL 8.0.27 官网下载地址：https://dev.mysql.com/downloads/mysql/
  2. 下载免安装版本到指定位置（建议下载在C盘以外的地方）

执行安装命令

  1. 在下载位置的bin文件夹下以管理员权限打开terminal
  2. 执行MySQL安装命令mysqld --install，出现Service successfully installed.时表示安装成功。

初始化MySQL及启动服务

  1. 执行初始化MySQL命令mysqld --initialize --console（执行大概需要3分钟），此时会产生一个临时密码，一定要记录，建议使用txt存储在电脑上
  2. 执行开启mysql的服务命令net start mysql

验证登录及修改密码

  1. 验证登录，执行mysql -u root -p，接着输入刚才生成的临时密码，出现欢迎提示则说明MySQL服务启动成功：
  2. 修改密码，在刚刚的命令行中输入alter user 'root'@'localhost' identified by 'new password';
  3. 验证新密码，在mysql命令行中输入exit退出，再输入mysql -u root -p，使用修改后的新密码登录，可以看到，登陆成功了。

SQL语句

  • SQL语句可以单行或多行书写，但一定以分号结尾。
  • SQL语句对大小写不敏感，但关键字建议使用大写。

注释

  • 单行注释：-- 单行注释或# 单行注释，注意中间的空格不能省略
  • 多行注释：/* 多行注释，可分行 */

SQL语句分类

  • DDL（Data Definition Language）：数据定义语言，操作数据库，数据表，列属性等
  • DML（Data Manipulation Language）：数据操作语言，对表中数据进行增删改
  • DQL（Data Query Language）：数据查询语言，对表中数据进行查询
  • DCL（Data Control Language）：数据控制语言，对数据库进行权限控制
