122. Best Time to Buy and Sell Stock II

You are given an array prices where prices[i] is the price of a given stock on the ith day.

You want to maximize your profit by choosing a single day to buy one stock and choosing a different day in the future to sell that stock.

Return the maximum profit you can achieve from this transaction. If you cannot achieve any profit, return 0.

Example 1:

Input: prices = [7,1,5,3,6,4]
Output: 5
Explanation: Buy on day 2 (price = 1) and sell on day 5 (price = 6), profit = 6-1 = 5.
Note that buying on day 2 and selling on day 1 is not allowed because you must buy before you sell.

Example 2:

Input: prices = [7,6,4,3,1]
Output: 0
Explanation: In this case, no transactions are done and the max profit = 0.

Constraints:

  • 1 <= prices.length <= 10^5
  • 0 <= prices[i] <= 10^4

Analysis:

  • 数据规模：1e5，因此只能使用一维数组，考虑dp[i] 的含义
  • 我们可以令 dp[i] 为第 i 天结束后，能够持有的最大金额
  • 考虑到第 i 天结束后，存在持有股票和不持有股票两种状态，因此我们可以考虑将 dp[i] 升级为 dp[i][2]
    • dp[i][0] 表示第 i 天结束后且不持有股票（股票数为0）的最大金额，
    • dp[i][1] 表示第 i 天结束后且持有股票（股票数为1）的最大金额，
  • 这样，状态转移方程就很容易得到
    • 第 i 天结束后没有股票，可能是昨天结束后本来就没有，或者是昨天结束后本来有，今天给卖掉了，即dp[i][0] = max(dp[i-1][0], dp[i-1][1]+prices[i])
    • 第 i 天结束后拥有股票，可能是昨天结束后本来就有，或者是昨天结束后本来没有，今天购买了，即dp[i][1] = max(dp[i-1][1], dp[i-1][0]-prices[i])
  • 最终的答案应该是第 n-1 天结束后且未持有股票的最大金额，因此不用记录全局最大值
class Solution {
public:
  int maxProfit(vector<int>& prices) {
    int n = prices.size();
    vector<vector<int>> dp(n, vector<int>(2));
    // initialize
    dp[0][0] = 0;
    dp[0][1] = -prices[0];
    for (int i = 1; i < n; i++) {
      dp[i][0] = max(dp[i-1][0], dp[i-1][1]+prices[i]);
      dp[i][1] = max(dp[i-1][1], dp[i-1][0]-prices[i]);
    }
    return dp[n-1][0];
  }
};

Optimization:

  • 通过状态转移方程可知，dp[i][j] 只依赖dp[i-1][j]，因此考虑只使用两组变量
class Solution {
public:
  int maxProfit(vector<int>& prices) {
    int n = prices.size();
    int preDay[2];
    int curDay[2] = {0, -prices[0]};
    for (int i = 1; i < n; i++) {
      int nextDay[2] = {
        max(curDay[0], curDay[1]+prices[i]),
        max(curDay[1], curDay[0]-prices[i]),
      };
      preDay[0] = curDay[0], preDay[1] = curDay[1];
      curDay[0] = nextDay[0], curDay[1] = nextDay[1];
    }
    return curDay[0];
  }
};

贪心

  • 思想：只要当前的股票价格大于昨天的股票价格，我们就在昨天买入，今天卖出，对于[1, 2, 3]，正确的买卖方式为买 1， 卖3，但我们买1卖2，买2卖3，可以视作买1卖3，这样，我们使用了两次购买代替正确的一次购买，在本题，不限购买次数时是等价的
class Solution {
public:
  int maxProfit(vector<int>& prices) {
    int n = prices.size();
    int maxProfit = 0;
    for (int i = 1; i < n; i++) maxProfit += max(0, prices[i]-prices[i-1]);
    return maxProfit;
  }
};
