Rendering Elements

React 元素(React Element) 是 React 应用的最小构建块，一个元素中包含你对网页呈现的描述，如文字为Hello，world的H1标题：

const element = <h1>Hello, world</h1>;

与浏览器 DOM 不同的是，React 元素是一个纯对象，创造成本低，并且 React 会负责更新 DOM 来保证符合期望的描述。

比起元素(element)，我们应该对组件(component)更加熟悉，关于组件，我们会在下一章进行讲解，实际上，组件是由元素所构成的，因此我们鼓励在学习组件之前，先了解元素的概念。

Rendering an Element into the DOM

首先在HTML文件中创建一个div标签，作为 React 的入口：

<div id="root"></div>

依照惯例，我们将该div的id设置为 root，称其为“根节点”，因为该 DOM 下的所有内容均由 React DOM 进行管理。

对于一个纯 React 应用，一般仅含有一个根节点；
如果是在现有的网页中渐进的引入 React，则一般每个引用 React 的地方都会存在一个根节点。

为了将 React 元素渲染（render）到 根节点上，将这两个作为参数传递给ReactDOM.render()：

const element = <h1>Hello, world</h1>;
ReactDOM.render(element, document.getElementById('root'));

整体的代码如下所示：

<!DOCTYPE html>
<html>

<head>
  <!-- React CDN -->
  <script src="https://cdn.staticfile.org/react/16.4.0/umd/react.development.js"></script>
  <!-- ReactDOM CDN 注意要放在 React CDN 下方 -->
  <script src="https://cdn.staticfile.org/react-dom/16.4.0/umd/react-dom.development.js"></script>
  <!-- babel CDN  用来解析JSX语法 -->
  <script src="https://cdn.staticfile.org/babel-standalone/6.26.0/babel.min.js"></script>
  <title>Hello React!</title>
</head>
<body>
  <!-- React 根节点 -->
  <div id="root"></div>
</body>

<!-- type="text/babel" 表示使用 babel 解析 -->
<script type="text/babel">
  const element = <h1>Hello, world</h1>;
  ReactDOM.render(element, document.getElementById('root'));
</script>

</html>

Updating the Rendered Element

React 元素是不变的（immutable）一但创建了元素，其属性和孩子节点就不能再改变，一个元素就像是电影中的一帧，代表了 UI 在某个时间的确定状态。

以目前的知识，我们更新元素的方法只有一种：创建新元素并再次调用ReactDOM.render()函数进行重新渲染：

<head>

React Only Updates What’s Necessary
