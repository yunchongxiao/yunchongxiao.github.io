Rendering Elements

React 元素(React Element) 是 React 应用的最小构建块，一个元素中包含你对网页呈现的描述，如文字为Hello，world的H1标题：

与浏览器 DOM 不同的是，React 元素是一个纯对象，创造成本低，并且 React 会负责更新 DOM 来保证符合期望的描述。

比起元素(element)，我们应该对组件(component)更加熟悉，关于组件，我们会在下一章进行讲解，实际上，组件是由元素所构成的，因此我们鼓励在学习组件之前，先了解元素的概念。
