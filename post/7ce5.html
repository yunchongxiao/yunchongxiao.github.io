42. Trapping Rain Water

Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it can trap after raining.

Example 1:

Input: height = [0,1,0,2,1,0,1,3,2,1,2,1]
Output: 6
Explanation: The above elevation map (black section) is represented by array [0,1,0,2,1,0,1,3,2,1,2,1]. In this case, 6 units of rain water (blue section) are being trapped.

Example 2:

Input: height = [4,2,0,3,2,5]
Output: 9

Constraints:

  • n == height.length
  • 1 <= n <= 2 * 10^4
  • 0 <= height[i] <= 10^5

Analysis:

  • 数据规模：1e4，因此只能使用一维数组，因此考虑dp[i]的含义
  • 对于某个位置 i 的坑，能够接到的雨水与其左右两端的最高墙的高度有关系，只有两边的最高墙均高于当前位置，当前坑才能够接到水
    • 进一步的，（水量为左、右两端最高墙高度的最低值 - 当前高度）× 坑的宽度（本题为 1）
  • 因此考虑dpl[i] 表示位置 i 左边的最高墙高度（包含i），dpr[i] 表示位置 i 右边的最高墙高度（包含 j）
  • 状态转移，欲求 dp[i]，则已知 dp[i-1] height[i]，i 位置的左边最大高度一定是他自己，或者 i-1 位置左边的最大高度，因此 dp[i] = max(height[i], dp[i-1])
  • 初始化：第 0 个位置的左边最高墙高度只能为他自己，第 n-1 个位置右边的最高墙高度也只能为他自己
class Solution {
public:
  int trap(vector<int>& height) {
    int n = height.size();
    vector<int> dpl(n);
    vector<int> dpr(n);
    int ans = 0;
    dpl[0] = height[0];
    dpr[n-1] = height[n-1];
    for (int i = 1; i < n; i++) {
      dpl[i] = max(height[i], dpl[i-1]);
    }
    for (int i = n-2; i >= 0; i--) {
      dpr[i] = max(height[i], dpr[i+1]);
    }
    for (int i = 0; i < n; i++) ans += min(dpl[i], dpr[i]) - height[i];
    return ans;
  }
};

Double pointer：

  • 我们定义两个指针 left, right，用来指向左右两个位置，我们每次计算时需要移动某个指针到达下一个位置 cur，收集位置 cur 的雨水，我们用 leftMax, rightMax，用来表示 cur 位置左右两端的最大高度
  • 在初始状态下，left 指向 0， right 指向 n-1，显然这两个位置都不能接到雨水。下面我们考虑移动哪一个指针能够得到正确的雨水量。
  • 如果下一个位置 cur 能够接到水，则 cur 位置的水位的高度左右两端最大高度中，最低的那一端，而小于较高的那一端，因此我们考虑移动较低那一端的指针（假设为左指针，即 l++）
  • 此时，我们能遇到三种情况：
    1. height[i] = leftMax：当前位置 i 既不会接到雨水，也不会改变左右两端最大高度，直接跳过
    2. height[i] < leftMax：当前位置 i 可以接到雨水，则一定不会改变左边的最大高度，存储接到的雨水，并继续移动
    3. height[i] > leftMax：当前位置 i 无法接到雨水，但可以改变左边的最大高度，因此更新左边的最大高度，下一次则移动新的较低的那一端
class Solution {
public:
  int trap(vector<int>& height) {
    int n = height.size();
  	int left = 0, right = n-1, leftMax = height[left], rightMax = height[right];
    int ans = 0;
    // if left == right, then end
    while (left < right) {
      // move left pointer, inner code will contious move left, so need judge left < right
      while (leftMax <= rightMax && left < right) {
        left++; // move
        int cur = height[left]; // current height
        if (cur == leftMax) continue; // change nothing, continue
        else if (cur > leftMax) { // update leftMax, can't get water
          leftMax = cur;
          continue;
        }
        // get water, can't change leftMax
        else if (cur < leftMax) {
          ans += leftMax-cur;
        }
      }
      // move right pointer
      while (rightMax <= leftMax && left < right) {
        right--; // move
        int cur = height[right]; // current height
        if (cur == rightMax) continue;
        else if (cur > rightMax) {
          rightMax = cur;
          continue;
        }
        else if (cur < rightMax) {
          ans += rightMax - cur;
        }
      }
    }
    return ans;
  }
};
