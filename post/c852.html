343. Integer Break

Given an integer n, break it into the sum of k positive integers, where k >= 2, and maximize the product of those integers.

Return the maximum product you can get.

Example 1:

Input: n = 2
Output: 1
Explanation: 2 = 1 + 1, 1 × 1 = 1.

Example 2:

Input: n = 10
Output: 36
Explanation: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36.

Constraints:

  • 2 <= n <= 58

Analysis:

  • 数据归规模为 58，因此几维数组都可以，我们先思考最朴素的 dp[i] 表示 i 被拆分能得到的最大乘积
  • 对于一个新的整数 i，我们有两种拆分方式：
    1. 拆成 2 个数的和
    2. 拆成多个数的和
  • 若拆成两个数，a, b，则乘积为 a*b
  • 若拆成多个数，也就是 a + (b + c + d + … + z)，乘积为 a * (b * c * d … * n)，也就是 a * dp[b+c+…+d]
  • 初始化 dp[2] = 1
class Solution {
public:
    int integerBreak(int n) {
      vector<int> dp(n+1);
      // i is splited into 1 + (i-1), result = 1 * (i-1) = i-1
      for (int i = 2; i <= n; i++) dp[i] = i-1;
      for (int i = 3; i <= n; i++) {
        // split into two numbers
        for (int a = 1; a <= i/2; a++) {
          int b = i - a;
          dp[i] = max(a*b, dp[i]);
        }
        // split into over two numbers
        for (int a = 1; a < i-1; a++) {
          int rest = i - a;
          dp[i] = max(dp[i], a*dp[rest])
        }
      }
      return dp[n];
    }
};
